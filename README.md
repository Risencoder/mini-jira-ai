# Mini Jira AI

Fullstack task management application inspired by Jira.

## Stack

Frontend:
- React
- Vite
- Axios
- React Router

Backend:
- Node.js
- Express
- Prisma
- PostgreSQL (Supabase)
- JWT Authentication

## Features

- Authentication (register / login)
- Projects
- Tasks
- Comments
- Project members
- Kanban board
